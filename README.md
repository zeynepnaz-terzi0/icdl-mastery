# ICDL Mastery

An interactive, single-page study app for the **ICDL (International Computer Driving Licence)** certification — built with vanilla HTML, CSS, and JavaScript. No frameworks, no dependencies, just one file.

## Features

- **7 Modules** covering the full ICDL syllabus (Computer Essentials, Online Essentials, Word Processing, Spreadsheets, Presentations, Databases, IT Security)
- **Quick Review** — concise notes and key concepts per module
- **Lecture** — in-depth content with structured sections
- **Flashcards** — flip-card format for self-testing
- **Practice Quiz** — multiple-choice questions with instant feedback and explanations
- **Recall Mode** — timed mixed quiz across all modules with streak tracking
- **Progress tracking** — study completion bar across all modules
- Dark-themed, responsive UI

## Demo

## Usage

1. Clone or download the repo
2. Open `index.html` in your browser
3. Pick a module from the sidebar and start studying
